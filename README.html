Classes

  • AbstracSyntaxTree.py (large image): AbstracSyntaxTree and tree node classes, these build the AST we use.
  • MyListener.py: A listener which can build an AST when walking over a parse tree generated by SmallCParser.
  • TypeInfo.py: A class that contains information about the type of a variable or function.
  • SymbolTable.py: Our symbol table class which is used for scope and type checking.
  • ASTSymbolTableFiller.py: A class that fills in a symbol table
  • CompilerErrorHandler.py: A class in which errors generated in tree nodes, MyListener, ASTSymbolTableFiller or SymbolTable are collected. We use this to collect error and display them after/during compilation. It is also used to check errors in testing
  • testfiles/main.py: ASTTest and test classes, ASTTest provides the test classes with functions to test functionality easily
