Build & Test

Je kan gebruik maken van de 2 .bat bestanden om op windows eerst te builden (build.bat) en vervolgens alle testen uitvoeren met test.bat

Via python kan je de testen uitvoeren met python main.py test

Status

  1. Types char, float, int en pointer types ondersteund
  2. Import Header wordt correct geparsed, de 2 functies zijn beschikbaar in de global scope van het programma indien de import gegeven is. De omzetting naar P-code moet nog gebeuren.
  3. Reserved words We ondersteunen: if, else, return, while, for, break, continue en void + de bovenstaande types
  4. Local and global variables Klaar
  5. Comments Klaar
  6. Functions Worden geparsed en correct omgezet in AST, forward declaration nog niet ondersteund. Parameters kunnen by reference of by value worden doorgegeven.
  7. Arrays Worden omgezet in AST nodes, voor de rest nog niet veel rond gedaan.
  8. Conversions Optioneel, nog niet aan begonnen, nog geen concrete plannen voor.

Error analyse

Errors die we checken aan de hand van semantische analyse:

  • Double declarations
  • Undefined references (zowel naar functies als variables)
  • Verkeerde return type
  • Type incompatibiliteit in expressies
  • Functie argumenten type check en aantal check
